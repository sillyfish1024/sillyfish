PS白底图改为透明背景

发表于2021-03-16|更新于2021-03-16|ps
|阅读量:

前言

在捣腾Blog网站logo的时候，需要一张透明底色的png图标，冲了半天浪也尽不能遂意。便想到DIY，又多少触及到知识盲区，在有限的技能点内也只想到用PowerPoint制作特效文字 → 截图得到白底图 → Photoshop扣底这种贻笑大方的方式。但也尚可作一点记录。

工具

Adobe Photoshop 22.3.0 release

过程

使用Photoshop打开图片，并解锁图层

使用颜色范围选择器选择并删除背景

菜单栏 → Select → Color range → 取色器点击白色背景 → OK → Delete键去除背景

保存图片

后记

通过以上方式最终得到了透明底色的logo图片，但图片存在着阴影和白边，且有点模糊，效果如下：

这能忍吗？Never!!!

故最终还是在网上白嫖了一个logo。/(ㄒoㄒ)/~~

后期补充：
在取色器试选用白色highlights可以得到更为清晰的图片，可白边依旧去不掉，过程不再赘述。
